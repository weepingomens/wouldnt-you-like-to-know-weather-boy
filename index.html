Some quick notes:

#1 Similarly to the post templates, WO's phone templates use custom profile fields available in the UserCP to set the background and user icon. A minimum size of 300x600px is recommended for a phone background, though some templates will clip this image, and an icon of 100x100px or larger is recommended for a phone icon. override classes of [phonebackground] and [phoneicon] should still operate in the same way as the ones in the post templates.

#2 For funsies: if you included [<3] in your topic description and have a hex code entered for your current ship partner, your received [ msg2 ] messages will color with their accent, as demonstrated in this thread! Make those threads with your fellow arsonist slowburn bae extra special!
NOTE: a value for time for phone1 templates can be omitted by using [phone1b], [phone1bcall], and [phone1bdial] bbcodes instead. everything else should work as normal!
